9.13 档监控模式怎么选？🎚进阶

可逆 vs 全覆盖 vs 强制隔离 —— 按场景选档。

⭐ 推荐起始
  • 端点映射（默认）：改 Agent base_url，无证书，HTTP，可逆
  • 系统代理：装自签 CA 实现 HTTPS MITM，覆盖所有应用
  • 沙箱隔离：内核级强制约束，进程 100% 覆盖无旁路
💡 个人开发 → 端点映射；团队审计 → 系统代理；不可信工具 → 沙箱

9.2Tier 1 配置详情 🔧进阶

每档卡片内嵌完整配置面板，不用进二级菜单。

  • 反向代理端点：默认 127.0.0.1:19112，可改
  • 协议适配器：勾选支持的 LLM 协议（Anthropic / OpenAI / Gemini / Ollama）
  • 底部图解动态展示流量路径
